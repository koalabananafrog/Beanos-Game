1. You need the unity engine to run this project.

2. If you want to review the C# code, open the folder named "Assets", then the folder named "Scripts". You'll find the heart of the code right away, here youll see the largest and most complicated parts of the Beanos game scripts. Then you can also check out the other folders, "Data", "Cool" and "Boring". The "Data" folder has scripts about database, memory saving functions and file creation. The "Cool" folder, you'll find the scripts that are handling the in game obstacles, an example is the rocket scripts, they achieve 3d rocket mechanics in a 2d player world, and explodes (probably why I named this folder "Cool" when I was a teenager). The "Boring" folder handles privacy policy.

3. Download and play a finished built PC version of the game(Mobile exsists too): https://beanosmaker.itch.io/beanos
